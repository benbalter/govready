What is this?

This is a server running {{ project }} on a GovReady accreditation-ready virtual machine. Think of it like a virtual car that you can keep, delete, save, copy, or customize.

Where is it?

This GovReady server is hosted on Amazon's public cloud. It is an "ec2 micro instance" running on hardware in Amazon's "us-east-1" data center in Northern Virginia. See DevOps/Sec for more details.

Who can see this?

This server can be seen by anyone on the Internet. Customizing the server requires logging in with a password.

What's it cost?

This evaluation server running in the cloud is free to try for six hours. Buy more time or Download this virtual machine to try locally.

How secure is this?

This server is configured according to NIST Guidelines for locking down a server. See DevOps/Sec for more details.

What is GovReady?

GovReady is a non-profit that creates and shares blueprints for hardened, accreditation-ready configurations of innovative technology.
